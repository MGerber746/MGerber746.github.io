Math Teaching App

Github Mobile Repo

Github API Repo

Technology: Java (Android Studio), DjangoREST

This was my first big project in a collaborative environment. We had an entire semester to build an app. We decided to create a mobile math app that would help teach math to young children. It included gamification of simple math problems, accounts for both teachers and students, and the ability for teachers to assign problems to their students. It was developed using a DjangoREST api backend and Android (Java) as the frontend.

This project allowed me to develop my collaborative skills as well as the chance to develop my software design skills. We were only able to overcome our differences through communication skills and collaboration. Creativity was brought in to help us plan out our app in the best way we could.
