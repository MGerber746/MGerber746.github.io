Statistic Analysis of Bitcoin

Github Repo

Technology: Python

The idea of this statisical analysis is future price direction predictions: using the data to determine probable future price direction of cryptocurrency given data from one day. The goal is to be able to predict if bitcoin will increase or decrease on a day to day basis. By running multiple algorithms, including decision tree, random forest classifier, gradient boosting classifier, SVC and logistic regression, the best algorithm was logistic regression. All others overfit when they were scored. Logistic regression consistently comes out to a reasonably high accuracy score of about 0.793. Using Python scripting the results was a 79% accuracy on the next month in increase/decrease of prices in Bitcoin.

This project displays my problem-solving skills in the determination of how to accuractly modify data to recieve a useful result. It also shows my ability to collaborate with a team and critical-thinking skill.
